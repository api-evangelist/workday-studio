Julien Sirocchi will be presenting at Scala eXchange December 2014 Abstract Typeclasses provide a means of adding behaviour to existing types retroactively (known as ad-hoc polymorphism ). They can make libraries easier to use, more powerful and more generic. This presentation is a distilled narrative from the long journey discovering how to effectively use typeclasses in Scala.
