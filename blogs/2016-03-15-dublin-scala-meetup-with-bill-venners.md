Last week, Workday Dublin hosted a joint meetup for Scala Users Group and Functional Kats in our office in Dublin. We were delighted to have Bill Venners talking about asynchronous testing support in the upcoming release of Scala Test 3.0 With 3.0, ScalaTest will support testing in Scala.js, providing most of the functionality available to Scala developers. To properly support Scala.js, ScalaTest has been extended to include asynchronous testing styles which should make it much easier for developers to test their own async code in a non-blocking manner.
